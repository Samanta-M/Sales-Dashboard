# 📊 Sales & Financial Performance Dashboard 

## 📌 Project Overview
This project analyses sales and financial data to uncover key trends in revenue, profitability, and customer behaviour.  
An interactive Power BI dashboard was developed to support **data-driven decision-making** and business performance monitoring.


## 🛠 Tech Stack
 📊 Power BI Desktop – Main data visualisation platform used for report creation.
 📂 Power Query – Data transformation and cleaning layer for reshaping and preparing the data.
 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, dynamic visuals, and conditional logic.
 📝 Data Modelling – Relationships established among tables (resorts, snow, and data_dictionary) to enable cross-filtering and aggregation.
 📁 File Format – .pbix for development and .png for dashboard previews. 


## 📂 Data Source
 🛒 Superstore Dataset (Orders, Returns, People)  
 💰 Finance Dataset (Cost, Profit, Performance Metrics)  


## 🚀 Features & Highlights

### ❗ Business Problem
Organisations often lack a clear view of sales performance and profitability, leading to inefficient decisions and missed growth opportunities.



### 🎯 Goal of the Dashboard
 Track key KPIs: **Sales, Profit, Quantity**  
 Identify **Top-performing products & regions**  
 Analyze **Profitability trends**  
 Enable **Interactive business insights**  



### 📊 Dashboard Walkthrough

 🔢 **KPI Cards** → Total Sales, Profit, Quantity  
 📈 **Sales Trend** → Monthly & yearly performance  
 🌍 **Regional Analysis** → Sales & profit by region  
 🏆 **Product Performance** → Top & bottom products  
 📦 **Category Insights** → Category-wise contribution  
 🎛 **Filters/Slicers** → Dynamic analysis by region, category, date  



## 💡 Key Insights
 📍 High-performing regions contribute the majority of revenue  
 📉 Certain products generate low profit margins  
 📅 Seasonal trends impact sales performance  
 📊 Data reveals opportunities for cost optimisation and growth
